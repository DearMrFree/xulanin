# xulanin.com\n\nProduction site for **Xubie Snacks** — Nina Lux's artisan snack business — with the **Xuliani** AI concierge embedded on every page.\n\n**Live**: https://xulanin.com\n\nXuliani is a human-with-AI student enrolled at [School of AI](https://sof.ai). The widget on this site is sourced from sof.ai so the agent's brain updates as it trains — this codebase never has to redeploy for Xuliani to get smarter.\n\n## Stack\n\n- **Framework**: Next.js 16 (App Router, Turbopack)\n- **Styling**: Tailwind CSS v4\n- **Hosting**: Vercel\n- **Backend**: FastAPI on Fly.io\n- **AI Agent**: Xuliani (via sof.ai embed)\n- **E-commerce**: Built-in ordering system\n\n## Local dev\n\n```bash\nnpm install\nnpm run dev          # http://localhost:3000\nnpm run build        # production build\nnpm run lint         # eslint\n```\n\n## Contributing\n\nOpen a pull request against `main`.
